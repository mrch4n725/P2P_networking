Secure P2P (Work in Progress)
⚠️ Note: This project is currently under active development and is not yet functional.
This is a conceptual Python-based peer-to-peer (P2P) script designed to facilitate direct file transfers between users. The project aims to provide a lightweight, decentralized way to share data without relying on a middleman server.
🎯 Project Goals
Direct File Transfer: A core focus on establishing stable connections between a host and a client for seamless file sharing.
Integrated Chat: Future updates aim to include a built-in messaging feature, allowing peers to communicate in real-time during transfers.
User-Friendly Interface: Built using Python's tkinter (ttk) to provide a simple, frame-based GUI for managing connections.
Security & Integrity: Planned implementation of SSL encryption and SHA-256 checksums to ensure data remains private and uncorrupted.
🛠️ Current Status
The UI layout is currently being drafted, featuring separate modules for Server and Client modes. The underlying networking logic and threading protocols are still in the prototype phase and are being refined for stability.
