Secure P2P (Work in Progress)

⚠️ Note: This project is currently under active development and is not yet functional.
This is a conceptual Python-based peer-to-peer (P2P) script designed to facilitate direct file transfers between users. The project aims to provide a lightweight, decentralized way to share data without relying on a middleman server.

🎯 PROJECT GOALS:

Direct File Transfer: A core focus on establishing stable connections between a host and a client for seamless file sharing.
-Integrated Chat: Future updates aim to include a built-in messaging feature, allowing peers to communicate in real-time during transfers.
-User Friendly Interface: Built using Python's tkinter (ttk) to provide a simple, frame-based GUI for ease of access and concise functionality.
-Security & Integrity: Planned implementation of SSL encryption and SHA-256 checksums to ensure data remains private and uncorrupted upon standard password authentication.
-Global direct connection: Planned implementation to the base script to support connection between machines over the internet.

🛠️ CURRENT STATUS:
*The UI is being drafted*, featuring separate modules for Server hosting and Client connection. The underlying networking and threading protocols are still in the prototype phase and are being refined for stability.

_*First version of UI released under test branch, semi-functional._
